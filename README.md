# 專案名稱
Diswork

## 介紹
一個讓大家隨時討論、盡情聊的線上討論版！

## 功能
1. 討論版
   - 多元主題討論版

3. 活動行事曆功能
     - 即時建立事件
     - 查看未來行程

3. 新聞
     - 自動更新每日頭條新聞

5. 會員系統
     - 第三方登入，快速方便
     - 劃分會員等級，解鎖付費功能

5. 即時聊天互動
   

## 專案初始化
#### 虛擬環境:
```
$ pip install poetry
$ poetry init -n
$ poetry shell
$ poetry install
```
## Node.js
```
$ install node
$ npm init -y
$ npm run dev
```
```
$ make server
```

## 使用技術
- 前端：Tailwindcss/Alpine.js/HTMX
- 後端：Python Django
- 資料庫：mySQL
- 第三方串接：Google OAuth 2.0/LINE/藍新金流
- 部署：Heroku/AWS S3
